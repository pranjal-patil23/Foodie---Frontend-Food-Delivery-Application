# FoodHub---Frontend-Food-Delivery-Application
Foodie is a responsive food ordering web application built using HTML, CSS, and JavaScript. It features dynamic menu rendering, real-time food search, cart management with Local Storage, quantity updates, and a modern user-friendly interface across multiple pages.

## 🚀 Overview

Foodie is a modern and responsive food ordering web application built using HTML, CSS, and JavaScript. The platform allows users to browse food items, search dishes, manage their cart, and simulate an online food ordering experience through an intuitive and user-friendly interface.

## ✨ Features

✅ Responsive Design

✅ Dynamic Food Menu

✅ Add to Cart System

✅ Cart Quantity Management

✅ Local Storage Integration

✅ Login & Signup Interface

✅ About Us Page

✅ Contact Page

✅ Mobile Friendly UI
